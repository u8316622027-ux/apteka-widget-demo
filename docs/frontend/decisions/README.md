# Frontend Decisions

Каталог для коротких записей по фронтенд-решениям, прежде всего по новым зависимостям.

## Формат записи
- Путь: `docs/frontend/decisions/<YYYY-MM-DD>-<topic>.md`
- Минимум:
  - Контекст проблемы
  - Выбранное решение
  - Альтернативы и почему отклонены
  - Влияние на bundle/сложность поддержки

## Пример имени файла
- `docs/frontend/decisions/2026-03-04-carousel-library.md`
